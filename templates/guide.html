Guide to MessageGuru

What does it do?

MessageGuru is a Server-Wide messaging service. With the help of this utility you can store a note in server database which can be accessed by any one in the server with the access code. With each access code a unique note is stored.

How to leave a note?

I have given you, the user, ability to choose access code of your own choice but if it is already in the server it will throw an ugly error.

Using this feature is easy. Just choose a access code which can be similar to a password in a sense. Then, choose a title for your note and then write a note. For now only single line code is supported.(i.e without line breaks)

How to recieve a message

To recieve a message you need access code. This can be provided by the one who created the message.

Input the case-sensitive access code and click on submit. you will then recieve the code.
